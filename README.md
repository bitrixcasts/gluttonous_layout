# Обжорозаврик

Верстка учебного проекта по фреймворку 1С-Битрикс. Для курса обучающих курсов проекта **Bitrixcasts**.

### Youtube

Ссылка на [Youtube канал](https://www.youtube.com/channel/UCXsxC7FoQclWTy9PNpy0ANQ)


### Запустить проект

```
npm install
```

А затем

```
npm run start
```
